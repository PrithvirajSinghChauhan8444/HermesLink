# HermesLink Project Root

This is the root directory for the HermesLink project.

## Files

- `config.yaml`: Configuration file for application settings (paths, engine options).
- `.env`: Environment variables file.
- `HermesLink_blueprint.txt`: The original development blueprint/roadmap.

## Directories

- `src/`: Source code directory.
- `test_downloads/`: Default output directory for downloads during testing.
- `.venv/`: Python virtual environment (ignored in documentation).
