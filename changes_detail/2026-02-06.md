# Changes Detail - 2026-02-06

## Linux Compatibility Fixes

The system is now **somewhat stable on Linux**. Multiple fixes were made to ensure the backend runs properly on Linux systems.

### Backend Fixes

- **REMOVED** Windows-only `import msvcrt` from `controller.py` - this module doesn't exist on Linux and wasn't being used.
- **ADDED** Missing `List` import to `core/models.py` - was causing `NameError: name 'List' is not defined`.
- **ADDED** Missing `Any` import to `engines/aria2.py` - was causing `NameError: name 'Any' is not defined`.
- **FIXED** Hardcoded Windows path `G:\STUFF\Watch\hermeslink_test_download` in `controller.py` - now uses cross-platform `~/Downloads/hermeslink_downloads`.

### Dashboard Improvements

- **FIXED** Non-blocking keyboard input for Linux using `tty.setraw()` with select timeout.
- **CHANGED** Key matching to use `.upper()` for case-insensitive input (M/m and Q/q both work).
- **ADDED** Enter key handling - pressing Enter now immediately refreshes the dashboard view.

### Aria2 Engine

- **VERIFIED** aria2c daemon starts and runs correctly on Linux.
- **TESTED** Download via aria2 engine completed successfully.
