# 2026-01-27

## Frontend Refactor: New Hub Layout

Moving to a new Frontend Layout designed as a "Hub & Spoke" model.

**Current Branch**: `frontend`

### Key Changes

- **Layout Architecture**:
  - Implemented a **Persistent Landing Hub**: The root path (`/`) acts as a launchpad.
  - **Prism 3D Background**: Integrated an interactive WebGL background on the Landing Page.
  - **Dynamic Positioning**: The main Title ("HermesLink") starts centered and moves up slightly to reveal the Navbar upon interaction, maintaining a centered "Hub" aesthetic.
- **Navigation**:
  - **New Window Strategy**: Dashboard pages (`Active`, `Queue`, `History`, `Settings`, `About`) now open in new windows/tabs (`target="_blank"`) to preserve the Hub state.
  - **Top Alignment in App**: When inside a feature page, the layout shifts to Top-Aligned to maximize content space.
- **Configuration**:
  - Updated `.gitignore` to exclude `_metadata.json` and `package.json`.
