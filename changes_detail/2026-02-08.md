## History Page Redesign

### Layout Changes
Converted History page to **25% / 75% split layout**:
- **Left Column (25%)**: Queue navigation sidebar with filters
- **Right Column (75%)**: Filterable history list

### Queue Filters
- All History
- Download Queue
- Processing Queue
- Completed
- Failed

### Liquid Glass Effect
Applied to all containers:
- Main split container
- Sidebar navigation
- List header
- History items

Glass effect properties:
```css
backdrop-filter: blur(12px) saturate(1.8) brightness(1.2);
box-shadow: 0 0 2px 1px light-dark(...) inset;
```

Added `Silk` component as animated background for History page.
