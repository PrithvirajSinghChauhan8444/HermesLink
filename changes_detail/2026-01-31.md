# Changes Detail - 2026-01-31

## Frontend Refactoring

### Navbar Placement
- **REFACTORED** `Navbar` to be included in each individual page (`LandingPage`, `ActiveJobs`, `QueueManagement`, `History`, `Settings`, `About`) instead of being a global component in `MainLayout` or `App.jsx`.
- **REMOVED** Global `Navbar` from `App.jsx`.
- **RATIONALE** This approach allows for per-page customization of the `Navbar`'s entrance animations and initial states.
