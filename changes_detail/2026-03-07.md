## Refactoring: Ferris Wheel Doodles 

Refined the visual appearance and dynamic behavior of the floating SVG doodles to act more like a rotating "Ferris Wheel" structure, creating a much more engaging and dynamic visual effect on scroll.

**Details:**
- **Ring Structure:** Arranged the floating doodles in a connected ring-like formation that rotates dynamically in response to user scrolling.
- **Independent Animations:** Ensured that individual doodles maintain their original autonomous up-and-down CSS animations while simultaneously rotating with the larger parent ring formation. 
- **Uneven Spacing & Fixed Radii:** Implemented randomized, uneven distances from the center for each doodle's position, using a specific fixed but irregular radius approach to produce a visibly more organic and interesting scattered effect around the rotating axle.
