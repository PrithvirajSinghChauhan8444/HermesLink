# Changes Detail - 2026-02-12

## Added "New Download" Section
- Implemented a "New Download" button in the `ActiveJobsSection` header.
- This opens a `NewJobModal` where users can input:
  - **URL**: Source link for the download.
  - **Type**: Download type (Direct, Media, P2P, Aria2).
  - **Queue**: Select target queue from backend configurations.
  - **Destination**: Optional custom path.

## Backend Updates
- Added `CreateJobRequest` model to `api_server.py`.
- Implemented `POST /api/jobs` endpoint to handle job creation.
- Updated default download type to `aria2` to align with preferred engine.
